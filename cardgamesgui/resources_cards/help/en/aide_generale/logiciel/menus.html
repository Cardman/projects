The software owns a menu bar with five menus.
These menus are:
  1. File, which lets gear
    1. files
    2. state changements of the software
  2. Deal, which lets
    1. gear a deal
    2. get help about way of playing for this deal
    3. train with certain situations of game
    4. see the possibilities of owning cards of the other players
    5. edit a deal
    6. learn playing by a demo of a deal
  3. Parameters, which lets gear
    1. rules of card games
    2. launching of the software
    3. nicknames of the players
    4. the timing of playing
    5. interaction between arrow of the mouse and cards
  4. Help, which gives help
