Load a deal (shortcut: CTRL + O)
The sub menu "Load a deal" opens a dialog box which lets load
any deal from this dialog box.

If a deal is being played, and if the deal has not been just saved, then
a warning message for the deal in progress appears.

In this case, the user has the choice between three options:
  1. Yes
  2. No
  3. Cancel
If the user has chosen "yes", then a dialog box for saving appears and
the user must give a name to the deal in progress.

If the user has chosen "no", then a dialog box of loading is opened.

If the user has chosen "cancel", then nothing happens.

The user can play or replay
a saved deal from a file. Besides, if the deal is edited, the user can play
the next deals if the user has chosen to play several consecutive deals.

Folders are grouped on a tree at the left of the dialog box.
Their content will be placed at the right of the dialog box, like Windows Explorer.

The button "Cancel" lets close the dialog box.

The user can search a file in the current folder with a text field above the content of folder.

If the user clicks a file of the enumerated list of files, then the dialog box is disposed.
Then:
  1. If the file is readable, then the deal (Either a simple deal or a set of deals,
      in the case when it is a set of deals, the loaded deal is the last played deal.), contained in this file, will be loaded.
  2. Else an error message appears displaying the loading fail.

Deals are checked about rules for avoiding bugs.
