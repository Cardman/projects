The deal of tarot is random and parameterized by rules.

It is possible to put in pause the deal for understanding the progress of the deal.

Differences between a deal the user plays and a demonstration are:
  1. No buttons interacting with the deal.
  2. In the text area, all information about the demonstrated deal is displayed.
  3. There is no dialog box for a demonstrated deal.
  4. The option consulting is always disabled.
  5. It is impossible to save the demonstrated deal.
  6. The demonstrated deal can be suddenly stopped.
