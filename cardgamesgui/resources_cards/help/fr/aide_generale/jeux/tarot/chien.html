Un chelem est une annonce où, l''annonceur (le joueur qui l''a annoncé) doit faire tous les plis, c''est-à-dire ramasser les cartes de ses adversaires.
Pour pouvoir l''annoncer, il faut demander un contrat supérieur ou égal à la Petite.

  1. Si le preneur a demandé une petite ou une garde:
      Le chien est dévoilé à tous les joueurs.
      Le preneur ramasse, ensuite, le chien.
      Il retire alors de son jeu le même nombre de cartes que le chien contient.

      Le preneur ne peut pas retirer d''atouts(L''Excuse est, ici, un atout.) ou de rois.
      Pour les atouts, si le preneur a un total d''atouts(L''Excuse est, ici, un atout.) et de rois(atouts + rois)supérieur au nombre de cartes par joueur,
      alors il montre à la défense les atouts qu''il veut écarter,
      dans ce cas, il peut enfin les retirer de son jeu.

      Les cartes retirées de son jeu sont face cachée pour la défense.
      Après écart de cartes dans le chien, le preneur peut annoncer un chelem.
  2. Si le preneur a demandé une garde sans ou une garde contre:
      Le chien n''est pas dévoilé.
      Si le preneur a demandé une garde sans, alors le chien appartient au preneur.
      Si le preneur a demandé une garde contre, alors le chien appartient à la défense.
      Le preneur peut annoncer un chelem.
