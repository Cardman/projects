Le logiciel possède une barre de menus avec cinq menus.
Ces menus sont:
  1. Fichier, qui permet de gérer
    1. les fichiers
    2. les changements d''état du logiciel
  2. Partie, qui permet de
    1. gérer une partie
    2. obtenir de l''aide sur la façon de jouer pour cette partie
    3. s''entraîner à certaines situations de jeu
    4. regarder les possiblités de possession de cartes des autres joueurs
    5. éditer une partie
    6. apprendre à jouer via une démonstration d''une partie
  3. Paramètres, qui permet de gérer
    1. les règles de jeux de cartes
    2. le lancement du logiciel
    3. les pseudonymes des joueurs
    4. la temporisation des jeux
    5. l''interaction entre la flèche de la souris et les cartes
  4. Aide, qui donne de l''aide
