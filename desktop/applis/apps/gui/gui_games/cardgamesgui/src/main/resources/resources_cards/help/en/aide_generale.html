This software lets play with two card games
which are belote and tarot and president.
Games and software are distinguished in the help file for being more readable.

The software is made for people who like playing cards, do not want download
card games everywhere on internet, and do not want to pay for playing
a card game.

The user can load a deal or save a deal.
The user can edit possible deals with different variants.
Parameters of game and launching software can be changed. The use can play with other people.
