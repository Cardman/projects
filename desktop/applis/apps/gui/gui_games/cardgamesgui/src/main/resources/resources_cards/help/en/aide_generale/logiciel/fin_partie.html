Displaying end of deal is done on the window,
by a tabbed pane, so no dialog box.
Tabs are the following one:
  1. Results of this deal (previous scores and scores of this deal).
  2. Detail results.
  3. Temporal graph.
  4. Hands and tricks.
The user can see previous scores, scores of this deal and declaring of this deal.

If the deal is finished at least for the second time, the deal is not counted.
Scores of random deals are separated from scores of edited deals.

The user can see hands and tricks of this deal.

Stack of all cards is saved in a file.
The number of deals played since the launch of the software is incremented (increased by one),
 then this number is saved in a text file.
The number of times, this deal is finished, is incremented.

Options of this state are the following one:
  1. "Keep playing the edited deals", if the finished deal is edited and not final.
  2. "Keep playing the deals", if the finished deal is edited and final and previously,
    at least one random deal has been played; or if the previous deal is random.
  3. "Replay the deal".
  4. "Stop".

If the user has chosen the option "Keep playing the edited deals", then cards are not mixed and the next deal is played.

If the user has chosen the option "Keep playing the deals", then the next deal is played.

If the user has chosen the option "Replay the deal", then the deal is redealt.

If the user has chosen the option "Stop", then the "single mode" menu with games is shown.
