Belote shortcut MAJ + B
The action of the sub menu "Belote" lets change variants of playing belote.

This action builds a dialog box.

The dialog box lets choose variants.
This does not change game rules of the possible played deal.
Game rules is changed only from the next deal played randomly or by a training.
Game rules is not changed while a competition.

The button, whose name is "Validate", is useful for validate modifications.

If the user has clicked the red cross at the top right of the window,
then no modification is done and the dialog box is disposed.
