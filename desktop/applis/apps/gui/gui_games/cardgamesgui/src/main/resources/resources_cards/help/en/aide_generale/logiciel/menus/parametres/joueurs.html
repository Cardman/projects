Players shortcut CTRL + ALT + J
The action of the sub menu "Players" lets change the nickname of the user and the nickname of the other players.

This action shows a dialog box.

The dialog box lets change:
  1. The nickname of the user.
  2. The nickname of the possible other players.

Warning, only the first nicknames, for the other players, will be used.

Example: if the game is played by 4 players, then only the 3 first nicknames
will be used.
