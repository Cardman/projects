Belote
The action of a sub menu "Edit" lets made some deal of belote in order that
the user test its way of playing and the user returns errors of playing to the author.

This action opens a dialog box.

The dialog box is introduced under two different forms:
  1. Choice of the number of players and rules of the belote before opening.
  2. Choice of the dealing of cards.

 Choice of the rules of the belote, for variants, influence on the dealing of cards
(number of hands, number of cards). Variants are described in the description of the belote.

 After having correctly dealt cards, (All hands are full at belote)
the user can save the deal in a file and possibly play it.
