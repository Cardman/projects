Ce logiciel est un programme qui permet de jouer à deux jeux de cartes
qui sont la belote, et le tarot, et le président.
On distingue les jeux du logiciel dans le fichier d aide pour être plus lisible.

Il est fait pour des gens aimant jouer aux cartes, ne voulant pas télécharger
les jeux de cartes un peu partout sur internet, et ne voulant pas payer pour pouvoir jouer
à un jeu de cartes.

L''utilisateur peut charger une partie ou sauvegarder une partie.
Il peut éditer toutes les parties possibles avec les différentes variantes.
Les paramètres de jeu et de lancement de logiciel peuvent être changés. Il existe
un mode multijoueur.
