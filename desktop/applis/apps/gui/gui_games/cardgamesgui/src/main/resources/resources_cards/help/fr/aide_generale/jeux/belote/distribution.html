Le joueur qui distribue les cartes s''appelle le donneur.
D''une partie à l''autre, le donneur tourne,
c''est à dire que le nouveau donneur est le joueur placé à droite de l''ancien donneur,
et le jeu est mélangé et coupé à chaque fois.

A la belote classique,
chaque joueur possède, au départ, 5 cartes. Le reste des cartes (en l''occurence 12) constituent un talon,
les cartes sont distribuées 3 par 3, puis 2 par 2.

Une fois que tous les joueurs ont cinq cartes, la carte au-dessus du talon doit être retournée.

A la belote coinchée,
chaque joueur possède 8 cartes,
les cartes sont distribuées 3 par 3, puis 2 par 2 pour les dernières.

Le donneur distribue dans le sens contraire des aiguilles d''une montre.
