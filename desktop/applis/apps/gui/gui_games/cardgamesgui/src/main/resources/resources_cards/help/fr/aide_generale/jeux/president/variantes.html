Comme pour tous les jeux de cartes, il existe des variantes.

Voici quelques variantes:

  1. Égaler est intedit: tous les joueurs ne peuvent pas jouer des groupes de cartes de même force que le dernier groupr.

  2. Égaler saute toujours le joueur suivant: si deux groupes of même force sont joués, le joueur suivant est sauté, (ce joueur ne peut pas jouer à ce moment, mais peut après).

  3. Égaler force le joueur suivant à égaler encore: si deux grouprs de même force sont joués, le joueur suivant doit égaler ou passer, (ce joueur peut seulement jouer un groupe de même force à ce moment, mais peut jouer n''importe quel groupe après). Si toutes les cartes de même force sont successivement jouées dans le pli, li pli est fini comme le jeu d''un groupe de 2.

  4. Tous les joueurs doivent jouer si possible. (ne peut pas passer)

  5. Si un joueur finit par un groupe de 2, le joueur peut perdre la donne. (Si deux joueurs finissent par un groupe de 2, le dernier joueur perd.)

  6. Si 4 cartes sont jouées en même temps, l''ordre est inversé (Les 2 deviennent faibles et les 3 deviennent forts.). Si 4 autres cartes sont jouées en même temps, l''inversion est annulée.

  7. Pour les donnes suivantes, le joueur au dernier rang échange 2 cartes avec le joueur au premier rang et le joueur à l''avant dernier rang échange 1 carte avec le joueur au deuxième rang.
    Les perdants donnent leur meilleures cartes. Les gagnants donnent des cartes de leur choix.

  8. Pour les donnes suivantes, le joueur au dernier rang commence le premier pli.

  9. D''autres jeux de cartes peuvent être ajoutés.

  10. Les cartes peuvent être distribuées dans le sens des aiguilles d''une montre.
