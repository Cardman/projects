Le tarot se joue avec 78 cartes, de 3 à 5 joueurs.
Il existe différents modes de jeu.
Nous allons présenter le mode de jeu classique qui est le jeu de contrat.
Le but, c''est de marquer le plus de points possible.

Plusieurs variantes (ou modes) existent. A l''origine, ce jeu se jouait à 4 joueurs.
Comme beaucoup de jeux de cartes, le tarot se joue carte par carte, les joueurs jouent une carte par tour.
