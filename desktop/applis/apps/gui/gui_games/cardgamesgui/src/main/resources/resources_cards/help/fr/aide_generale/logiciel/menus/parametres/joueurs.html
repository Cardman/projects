Joueurs raccourci CTRL + ALT + J
L''action du sous menu "Joueurs" permet de changer le pseudo de l''utilisateur et des autres joueurs.

Cette action met en place une boîte de dialogue.

La boîte de dialogue permet de changer:
  1. Le pseudonyme de l''utilisateur.
  2. Le pseudonyme des autres éventuels joueurs.

Attention, seuls les premiers pseudonymes,pour les autres joueurs, seront utilisés.

Exemple: si le jeu s''effectue à 4 joueurs, alors seuls les 3 premiers pseudonymes
seront utilisés.
