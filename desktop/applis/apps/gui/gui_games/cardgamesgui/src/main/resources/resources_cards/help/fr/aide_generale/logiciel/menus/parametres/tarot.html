Tarot raccourci MAJ + T
L''action du sous menu "Tarot" permet de changer les variantes de jeu pour le tarot.

Cette action met en place une boîte de dialogue.

La boîte de dialogue permet de mettre en place des variantes.
Ceci ne change pas les règles du jeu de l''éventuelle partie jouée.
On change les règles du jeu du tarot à partir de la prochaine partie de cartes aléatoire ou d''entraînement.
On ne change pas les règles du jeu du tarot lors d''un tournoi.

Le bouton, dont le nom est "Valider", sert à valider les modifications.

Si l''utilisateur a cliqué sur la croix rouge en haut à droite de la fenêtre,
alors aucune modification n''est faite et la boîte de dialogue se ferme.
