Temporisation du jeu raccourci F4
L''action du sous menu "Temporisation du jeu" permet de changer
  1. les délais entre deux cartes consécutivement jouées par les "robots".
  2. les délais entre deux plis ou tours consécutivement joués.
  3. les délais entre deux contrats consécutivement joués par les "robots".

Cette action met en place une boîte de dialogue.

Cette boîte de dialogue permet de fixer les délais entre:
  1. Deux cartes consécutives.
  2. Deux contrats consécutifs.
  3. Deux plis consécutifs.

De plus, cette boîte de dialogue permet
d''activer ou de désactiver les délais entre deux plis consécutifs,
sans cliquer sur les boutons de la fenêtre placés en bas à droite de la fenêtre.
