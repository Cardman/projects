Conseil Raccourci F1
L''action du sous menu "Conseil" permet d''obtenir de l''aide pour
  1. Annoncer un contrat
  2. Annoncer une enchère
  3. Annoncer une carte
  4. Ecarter une ou plusieurs cartes
  5. Jouer une ou plusieurs cartes

Cependant, il n''y aura pas d''aide pour les jeux solitaires.

Cette boîte de dialogue permet de conseiller l''utilisateur, sans l''obliger,
d''annoncer un contrat ou une enchère, d''appeler une carte,
de jouer une carte dans un pli, d''écarter des cartes dans un talon
ou d''échanger des cartes.

On donne la raison de l''action conseillée.
